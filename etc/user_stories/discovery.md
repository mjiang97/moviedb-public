# Movie/TV Shows Tracker

## **Persona**
- **Name:** Jordan
- **Relevant Characteristics:** [Background, role, experience level]
    College student, tech literate but not techsavvy 
- **Behaviors & Attitudes:** [How they use technology, frustrations, goals]
    Indecisive, gets overwhelmed choosing what to watch
- **Goal:** [What they want to achieve]
    They want an application that can help get recommendations that match interests
---

## **User Story**
[Write the user story using one of the approved formats]

As an indecisive viewer,
I want to get personalized movie and TV show recommendations,
so that I can easily find something new to watch without endlessly scrolling.

## **Acceptance Criteria**
- [Define the success conditions for this feature]
    - Aesthetically pleasing: pastel vs vibrant, neutral color palette? TBD
    - Easily accessible on PC/mobile
    - Can view curated lists
    - Can shuffle/refine lists

- [How will we know when this is done?]
    - Users can view curated lists

- [What must be functional before release?]
    - Ability to add recommended titles to a user's watchlist
    - A simple, user-friendly interface for browsing recommendations
